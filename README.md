# RoutineTaskManager

RoutineTaskManager - Android-приложение для планирования ежедневной рутины, напоминаний и рабочих сессий.

> Проект находится в процессе разработки. Некоторые экраны, сценарии задач и логика напоминаний могут быть не завершены или измениться.

## Возможности

- Создание и редактирование напоминаний с гибким расписанием повторов.
- Уведомления для обычных напоминаний через Android alarms.
- Отслеживание статусов напоминаний.
- Рабочие сессии с отдельными сессионными напоминаниями и foreground-таймером.

## Стек

- Kotlin
- Jetpack Compose and Material 3
- AndroidX Navigation 3
- Room
- Koin
- Coroutines and Flow
- Kotlinx Serialization
- Coil

## Структура проекта

- `featureReminder` - повторяющиеся рутинные напомнимания, расписание, уведомления, рабочие сессии и UI напоминаний **(в процессе разработки)**.
- `featureHome` - главный экран с расписанием на сегодня и управлением рабочей сессией **(в процессе разработки)**.
- `featureTask` - одноразовые напоминания **(в процессе разработки)**.
- `featureWidgets` - виджеты-трекеры **(в процессе разработки)**.
- `core` - общие уведомления, ошибки, UI-компоненты, сериализация и утилиты.
- `data` - локальная база данных и хранение файлов.

## Заметки

Приложение использует разрешения на уведомления, точные будильники, boot receiver и foreground service для доставки напоминаний и работы таймера сессии.
